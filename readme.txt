== Description ==

The plugin helps to install and customize accessiBe — a web accessibility solution for ADA & WCAG compliance. After a user-friendly installation process, you will find accessiBe customization panel inside the WordPress dashboard. With its help, you can easily adjust accessiBe's interface — colors, position, font size, trigger icons etc.

Before, you needed to make the customization process out of WordPress, inside the accessiBe solution. The plugin makes it possible to work with accessiBe and your website settings in one place — your WordPress dashboard.
